# 🎮 Split.gg

Split.gg é um projeto web feito em React para cadastro de jogadores e balanceamento automático de times.

## Funcionalidades

- Cadastro de jogadores
- Listagem e remoção de jogadores
- Seleção de jogadores para partida
- Balanceamento automático de times
- Histórico de partidas
- Persistência local usando `localStorage`

## Tecnologias

- React
- Vite
- JavaScript
- CSS
- LocalStorage

## Como rodar

```bash
npm install
npm run dev
```

Depois acesse o endereço exibido no terminal, normalmente:

```bash
http://localhost:5173
```

## Observação

Esta versão abandonou a ideia mobile/Expo e foi reorganizada como aplicação web, seguindo uma estrutura mais parecida com projeto integrador.
